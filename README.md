# AI Catalog Intelligence Platform

Production-grade multimodal e-commerce catalog intelligence system.

## Features

* Multimodal category classification
* Image–title mismatch detection
* Duplicate product detection
* Confidence-based review queue
* Semantic embedding storage

## Tech Stack

* PyTorch
* OpenCLIP
* FAISS
* FastAPI
* Next.js
* SQLite
* MLflow
* DVC

## Dataset

Amazon Berkeley Objects (ABO) – curated 6-category subset.
